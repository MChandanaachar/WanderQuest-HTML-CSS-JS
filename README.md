WanderQuest – Explore Karnataka

WanderQuest is a simple travel website focused on exploring destinations across Karnataka.
It showcases popular places, travel experiences, and activities like trekking, wildlife safaris, water adventures, and cultural tours.

Features

Fixed navigation bar with smooth scrolling

Hero section with search-based navigation

About section describing the platform

Top destinations displayed using cards

Services section highlighting travel experiences

Contact form for user inquiries

Scroll-based animations and active navigation highlighting

Responsive design for desktop and mobile devices

Technologies Used

HTML5

CSS3

JavaScript (Vanilla)

Google Fonts (Poppins)

Download or clone the repository

No additional setup is required

Purpose

This project is built for learning and showcasing frontend development skills, including layout design, animations, and responsive UI using plain HTML, CSS, and JavaScript.
